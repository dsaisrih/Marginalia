Connect Marginalia to Firebase

This copy isn't connected to a Firebase project yet, so notes can't be saved. Open this HTML file in a code editor, find the firebaseConfig object near the bottom of the file, and replace it with your own project's config. Then enable Firestore Database and Authentication → Anonymous + Email/Password in the Firebase console for that project.

Example only — for reference. Editing this text does nothing.

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.appspot.com",
  messagingSenderId: "...",
  appId: "..."
};

👉 The real, working firebaseConfig lives near the bottom of the file, inside the <script type="module"> section — search for PASTE YOUR OWN FIREBASE CONFIG BELOW to jump straight to it.

Recommended Firestore rules (Console → Firestore → Rules):

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /notes/{noteId} {
      allow read, update, delete: if request.auth != null
        && request.auth.uid == resource.data.uid;
      allow create: if request.auth != null
        && request.auth.uid == request.resource.data.uid;
    }
  }
}
